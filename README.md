# Pluto_IIT_Ropar_Ch2

Code for Inter IIT Tech Challenge - 2

Use : Python 3.9+

Steps to run the code:

1.  Connect webcam and place it on the roof
2.  Run 
    ```
    py ./aruco_verify.py
    ```
    to adjust the camera and place to drone
3.  Run
    ```
    py ./main.py
    ```
    to perform Challenge 2.

Press ```e``` for emergency landing.
